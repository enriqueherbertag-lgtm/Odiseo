# Arquitectura General de Odiseo

## Estructura Multicapa

| Capa | Material | Función |
|------|----------|---------|
| **Exterior** | Tungsteno-vanadio (W-V) | Estructural, calor, micrometeoritos |
| **Espacio de servicio** | Vacío | Bobinas superconductoras, tuberías de hidrógeno |
| **Absorción** | HDPE + tanques de H₂ | Neutrones, fragmentos, radiación secundaria |
| **Cabina (núcleo)** | Aluminio compuesto | Presurizada, tripulación |
| **Rueda** | HDPE + revestimiento | Protección adicional donde vive la tripulación |
