# Energía de Odiseo

## Fuentes Principales

| Fuente | Función |
|--------|---------|
| **Reactor nuclear (NTP/NEP)** | Propulsión + generación eléctrica durante tránsito |
| **RTG nuclear (50 años)** | Energía de respaldo para sistemas críticos |
| **Paneles solares** | Generación adicional en núcleo y rueda |
| **Baterías de litio** | Almacenamiento y respaldo |

## Transferencia de Energía a la Rueda

- **Transferencia inductiva (sin contacto)** para pasar energía del núcleo a la rueda giratoria
- **Baterías propias en la rueda** como respaldo autónomo
