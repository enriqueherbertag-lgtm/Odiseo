# Producción de Recursos In-Situ (ISRU)

## Torres ShieldAir

Odiseo utiliza las torres **ShieldAir** (adaptadas para Marte) para producir oxígeno, biomasa y calor desde la atmósfera marciana (95% CO₂).

| Recurso | Producción por torre (estimada) | Uso |
|---------|--------------------------------|-----|
| Oxígeno (O₂) | 0.8 toneladas/día | Respiración, oxidante para Goliat-Son |
| Biomasa (algas) | 15 kg/día | Alimento, bioplásticos, fertilizantes |
| Calor | 60–80 kW térmicos | Calefacción, derretimiento de hielo |

## Proceso

| Etapa | Proceso | Producto |
|-------|---------|----------|
| 1. Captura | Compresores aspiran aire marciano (95% CO₂) | CO₂ concentrado |
| 2. Fotosíntesis | Algas + CO₂ + H₂O + luz → O₂ + biomasa | O₂, algas |
| 3. Licuefacción | Enfriamiento criogénico | O₂ líquido (LOX) |
| 4. Procesamiento de biomasa | Secado, prensado, fermentación | Proteína, lípidos, bioplásticos |
| 5. Recuperación de calor | Intercambiadores de calor | Calor para hábitats, derretimiento de hielo |

## Requerimientos por torre

| Recurso | Cantidad diaria | Origen |
|---------|-----------------|--------|
| CO₂ | 1.1 toneladas | Atmósfera marciana |
| Agua | 0.5 toneladas | Hielo subterráneo (extraído) |
| Energía eléctrica | 10–15 kW | Reactor nuclear de Odiseo o paneles solares |
| Luz LED | 8–10 kW | Energía eléctrica |
| Nutrientes | 1 kg | Traídos de Tierra (inicial), luego reciclados |

## Integración con Odiseo y Goliat-Son

| Proyecto | Uso de ShieldAir |
|----------|------------------|
| **Odiseo** | Energía nuclear alimenta las torres; calor recuperado se usa en la base. |
| **Goliat-Son** | Reabastece sus tanques de O₂ con LOX producido por las torres. |
| **Base marciana** | O₂ para respirar, calor para hábitats, biomasa para alimento/plásticos. |
