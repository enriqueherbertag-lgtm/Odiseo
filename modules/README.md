# Módulo de Carga y Transporte (MCT)

Este módulo es un aterrizador autónomo basado en **Goliat-Son**, un proyecto independiente diseñado para ser ajustable a diferentes misiones.

## Repositorio

🔗 [Goliat-Son - Aterrizador autónomo](https://github.com/enriqueherbertag-lgtm/Goliat-Son)

## Uso en Odiseo

En Odiseo, el MCT se utiliza para:
- Aterrizar a los 6 tripulantes en la superficie marciana.
- Desplegar la carga útil de 8.7 toneladas.
- Reabastecerse con combustible producido in situ y volver a órbita.

Para especificaciones técnicas completas y opciones de configuración, ver el repositorio independiente de Goliat-Son.
