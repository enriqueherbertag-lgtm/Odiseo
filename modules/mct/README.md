# Módulo de Carga y Transporte (MCT)

Este módulo es el aterrizador y carguero de Odiseo. Transporta 6 personas y hasta 8.7 toneladas de carga a la superficie marciana.

## Repositorio independiente

El MCT está basado en **Goliat-Son**, un aterrizador autónomo desarrollado como proyecto independiente.  
Para detalles técnicos completos (estructura, propulsión, sistemas), ver:

🔗 [Repositorio Goliat-Son](https://github.com/enriqueherbertag-lgtm/Goliat-Son)

## Funciones en Odiseo

| Función | Descripción |
|---------|-------------|
| **Transporte orbital** | Acoplado a Odiseo durante el tránsito Tierra-Marte. |
| **Aterrizaje** | Descenso autónomo a superficie marciana con 6 personas y carga. |
| **Despliegue de base** | Entrega hábitats, paneles solares, torres ShieldAir, etc. |
| **Reabastecimiento** | Puede recargar combustible (CH₄ + O₂) producido in situ y volver a órbita. |
